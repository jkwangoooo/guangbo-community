Logo exploration / first round

光波社群

这一轮把品牌关键词放在“光、波、共振、连接、路径”上。标志尽量保持清晰、直接、可信，避免课程海报感和过度装饰。

  • Format: SVG, viewBox 0 0 100 100
  • Color: currentColor based, easy to recolor
  • Use cases: avatar, favicon, H5 header, social poster

Six Distinct Directions

每个方向都保留了足够的留白和小尺寸可辨识性，但符号性格有明显区分，方便你挑选后继深化。

01 / Recommended

Radiant Core

中心光源加断开波环，最直接地表达“微光成炬”和稳定扩散。适合做主标。

02 / Path

Wave Path

用一条流畅的波线把几个节点串起来，更像业务路径和持续陪跑。气质更友好。

03 / Community

Resonance Dots

点阵形成从中心向外的共振场，社群感最强。适合做头像或群封面标识。

04 / Entry

Light Gate

把“入口”做成光门，强调低门槛、可进入、有引导。信任感比较强。

05 / Link

Orbit Link

轨道和中心节点表达连接、资源循环和稳定支点。更偏科技、资源型社群。

06 / Route

Prism Route

三角稳定结构加内部路径，表达把分散的光转成明确行动。更像项目体系。
