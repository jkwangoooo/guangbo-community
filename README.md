# 光波社群静态页

这是一个可直接发布到 GitHub Pages 的纯静态页面项目。

## 线上地址

- GitHub Pages: https://jkwangoooo.github.io/guangbo-community/
- 国内备用直链: https://cdn.jsdelivr.net/gh/jkwangoooo/guangbo-community@main/index.html

## 本地文件

- `index.html`
- `styles.css`

## 自动发布

仓库内已包含 GitHub Actions 工作流：

- 推送到 `main` 分支后，自动发布到 GitHub Pages
- 发布内容仅包含页面运行所需文件，不会把设计稿截图一起暴露到线上站点

## 国内访问建议

- 默认地址可使用 GitHub Pages
- 额外建议准备一个自定义域名，并把域名接到 GitHub Pages
- 如果只是临时分享，也可以补充一个 jsDelivr 直链作为备用入口
